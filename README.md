# Shapes

Shapes is a Python-based educational application designed to help children learn about different geometric shapes in a fun and interactive way. The app introduces kids to basic shapes, their names, and visual representations, making early learning engaging and enjoyable.

## Features
	•	Interactive Learning: Children can click on shapes to hear their names or see visual examples.
	•	Variety of Shapes: Includes common shapes like circles, squares, triangles, rectangles, and more.
	•	Kid-Friendly Interface: Simple, colorful, and easy to navigate for young learners.
	•	Educational Focus: Helps develop shape recognition and early geometry skills.
	•	Extensible: Easily add more shapes or features using Python.

## Installation
	•	Clone this repository: ---> git clone https://github.com/KodyKam/Shapes.git
	•	Navigate to the project directory: ---> cd shapes
	•	Install required dependencies: ---> pip install -r requirements.txt
	•	Run the application: ---> python main.py

## Usage
	•	Launch the app and choose a shape from the menu.
	•	Learn the shape by interacting with it through clicks or animations.
	•	Test your knowledge with simple quizzes (if implemented).
